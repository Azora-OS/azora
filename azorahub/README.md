## üöÄ Getting Started

### Prerequisites
- Node.js 18+
- Git
- Azorahub account (free)

### Quick Installation
```bash
# Install Azorahub CLI
npm install -g @azorahub/cli

# Configure your account
azorahub configure

# Create your first project
azorahub create my-awesome-project
```
