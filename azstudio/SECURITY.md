## Security

Azora takes security seriously. If you discover a potential security vulnerability, please do not report it through public GitHub issues.

To report a security vulnerability, please contact our security team via one of the options below:

- Email: security@azora-technologies.com
- Web: https://azora-technologies.com/security/report

We will respond as quickly as possible and provide a coordinated disclosure timeline where applicable.

For other security policies and guidance, visit our security page: https://azora-technologies.com/security
