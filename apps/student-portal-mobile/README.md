# Azora Student Mobile App

## Setup

```bash
npm install
npm start
```

## Run on Device

```bash
npm run android  # Android
npm run ios      # iOS
```

## Features Implemented

- ✅ Authentication with AsyncStorage
- ✅ Dashboard with stats
- ✅ Course listing
- ✅ Wallet & transactions
- ✅ Profile & logout
- ✅ API integration
- ✅ Offline token storage
- ✅ Pull-to-refresh

## API Endpoints Used

- POST /api/auth/login
- GET /api/student/dashboard
- GET /api/courses/enrolled
- GET /api/wallet/balance
