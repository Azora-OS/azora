# Personalization Engine

The Personalization Engine is a service for Azora OS that provides personalized experiences for users based on their preferences, behavior, and context.

## Features

- User preference analysis
- Content recommendation system
- Adaptive interface personalization
- Context-aware customization

## API Endpoints

- `GET /health` - Service health check
- `POST /api/analyze-preferences` - Analyze user preferences
- `POST /api/recommend-content` - Recommend personalized content
- `POST /api/personalize-interface` - Personalize user interface

## Environment Variables

- `PORT` - Service port (default: 3014)
