# Azora Enrollment Service

Course enrollment and registration management service.

## Features

- Course enrollment processing
- Payment integration
- Waitlist management
- Enrollment analytics

## API Endpoints

- `POST /api/enrollments` - Enroll in course
- `GET /api/users/:userId/enrollments` - Get user enrollments
- `GET /api/health` - Health check
