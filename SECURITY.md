# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 3.0.x   | :white_check_mark: |
| < 3.0   | :x:                |

## Reporting a Vulnerability
Email: security@azora.world
Response time: 48 hours
Disclosure: Coordinated after fix

## Security Measures
- MFA authentication
- OAuth 2.0 + JWT
- Rate limiting
- CORS protection
- Input validation
- SQL injection prevention
- XSS protection
