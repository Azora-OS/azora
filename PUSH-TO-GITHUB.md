### Option 1: Using GitHub CLI (Recommended)
```bash

# macOS: brew install gh

gh auth login

# Push
git push origin main --tags
```

### Option 2: Using SSH
```bash

git remote set-url origin git@github.com:Sizwe780/azora-os.git
```
