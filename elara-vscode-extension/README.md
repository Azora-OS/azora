## Installation

### Option 1: From Source (Recommended)
1. Clone the Azora-OS repository
2. Navigate to `elara-vscode-extension/`
3. Run `npm install`
4. Run `npm run compile`
5. Press `F5` to launch extension development host
6. Test the extension in the new window

### Option 2: VS Code Marketplace (Future)
The extension will be published to the VS Code marketplace for easy installation.

## Configuration

Configure Elara through VS Code settings:

```json
{
  "elara.serverUrl": "http://localhost:3000",
  "elara.apiKey": "",
  "elara.enableAutoComplete": true,
  "elara.enableInlineSuggestions": true,
  "elara.enableRealTimeCollaboration": false
}
```

## Requirements

- **VS Code**: ^1.74.0
- **Node.js**: ^16.0.0
- **Elara Server**: Running Azora OS with Elara services

## Server Setup

Elara requires the Azora OS backend to be running:

1. Start Azora OS: `npm run start-azora-systems`
2. Ensure Elara services are running on port 3000
3. The extension will automatically connect
