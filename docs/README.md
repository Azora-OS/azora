### How to Help

1. **Fix typos/errors** - Submit a PR
2. **Add examples** - Clarify concepts
3. **Update outdated info** - Keep it current
4. **Translate** - Help reach more people

### Process

2. Create a branch: `docs/your-topic`
3. Make changes
