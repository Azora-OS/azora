# Azora OS: Community Edition

Welcome to the open-source Community Edition of Azora OS. This repository contains the core operating system, basic AI routing, and fundamental blockchain components.

## Community vs. Azora Cloud

| Feature | Community Edition | Azora Cloud (Premium) |
|---------|-------------------|-----------------------|
| **AI Routing** | Local LLM + Direct External | Hierarchical (RAP System), Cost Optimization |
| **Predictive Engine** | Disabled (Stubs) | PrPEng (Predictive Prompt Engineering) |
| **Minting** | Fixed Rate / Testnet | Dynamic Mining Algorithms, Value Calculator |
| **Safety** | Basic Keyword Filter | Constitutional AI (Contextual Analysis) |
| **Infrastructure** | Standard Logging | Chaos Monkey, Compliance Engine |

## Getting Started

The Community Edition is fully functional for development, testing, and basic deployments.

### Prerequisites
- Node.js 18+
- Docker
- Redis (Optional, for basic caching)

### Installation
```bash
npm install
npm run build
npm start
```
