# Azora OS Privacy Policy

**Last Updated:** 2025-01-10

## 1. Data Collection

We collect the following data:
- Account information (email, name, profile)
- Learning progress and course data
- Financial transactions and wallet balances
- Usage analytics and system logs

## 2. Data Usage

Your data is used to:
- Provide educational services
- Process financial transactions
- Improve platform performance
- Ensure security and compliance

## 3. Your Rights (GDPR Compliant)

You have the right to:
- **Access**: Request a copy of your data
- **Rectification**: Correct inaccurate data
- **Erasure**: Request deletion of your data (Right to be Forgotten)
- **Portability**: Export your data in machine-readable format
- **Withdraw Consent**: Opt-out of non-essential data processing

## 4. Data Retention

| Data Type | Retention Period | Auto-Delete |
|-----------|------------------|-------------|
| Activity Logs | 90 days | Yes |
| Transactions | 7 years | No |
| Course Progress | 5 years | No |
| Session Data | 30 days | Yes |

## 5. Contact

For privacy requests: privacy@azora.world
