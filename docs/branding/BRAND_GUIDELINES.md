### Font Weights
- **Light (300):** Subtle text, less important content

- **Medium (500):** Navigation, emphasized text
- **Semibold (600):** Subheadings, card titles

## ðŸŽ¨ UI/UX GUIDELINES

### Design Principles

#### 1. Simplicity with Depth
Clean interfaces that reveal complexity when needed

#### 2. Constitutional Alignment
Every feature serves humanity and honors our values

#### 3. Accessibility First
Inclusive design for all abilities and contexts

#### 4. Performance Matters
Fast, responsive, efficient - respect user's time

#### 5. Beauty as Worship
Excellence in design glorifies the Creator

### Component Library
- **Buttons:** Rounded (8px), bold text, clear hierarchy
- **Cards:** 12px border-radius, subtle shadows
- **Forms:** Clear labels, helpful validation
- **Icons:** Consistent style, 24px default size
- **Spacing:** 8px base unit, multiples for consistency

### Animation Principles
- **Purposeful:** Every animation has meaning
- **Fast:** 200-300ms for most transitions
- **Smooth:** Ease-out for natural feel
- **Optional:** Respect reduced-motion preferences
