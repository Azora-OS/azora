# Support Guide

If you need help with Azora OS, please use the following channels:

- General inquiries: support@azora.world
- Security incidents: security@azora.world (response within 48 hours)
- Enterprise support: enterprise@azora.world
- Discussions and Q&A: GitHub Discussions

When reporting issues, please include:
- Steps to reproduce
- Expected vs actual behavior
- Logs and screenshots (if applicable)
- Affected service/package name and version

Thank you for helping us improve Azora OS.
