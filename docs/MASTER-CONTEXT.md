## Deployment

### Local Development

```bash
# Install dependencies
npm install

# Setup environment
cp .env.example .env

# Start Chronicle Protocol
cd services/chronicle-protocol
npm install && npm start

# Start Master System
cd ../..
```

**Frontends:** Vercel

```bash
npx hardhat deploy --network polygon
```
