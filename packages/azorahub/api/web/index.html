API Endpoints

Repositories

GET /v1/repositories
List all repositories accessible to the authenticated user
curl -X GET "https://api.azorahub.com/v1/repositories" \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "Content-Type: application/json"
POST /v1/repositories
Create a new repository
curl -X POST "https://api.azorahub.com/v1/repositories" \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "name": "my-new-repo", "description": "A new repository", "private": false }'
GET /v1/repositories/{id}
Get a specific repository by ID
curl -X GET "https://api.azorahub.com/v1/repositories/123" \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "Content-Type: application/json"

Issues & Pull Requests

GET /v1/repositories/{id}/issues
List issues and pull requests for a repository
curl -X GET "https://api.azorahub.com/v1/repositories/123/issues" \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "Content-Type: application/json"
POST /v1/repositories/{id}/issues
Create a new issue or pull request
curl -X POST "https://api.azorahub.com/v1/repositories/123/issues" \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "title": "Bug: Authentication fails", "body": "Detailed description of the issue", "labels": ["bug", "authentication"] }'

AI Services

POST /v1/ai/code-completion
Get AI-powered code completion suggestions
curl -X POST "https://api.azorahub.com/v1/ai/code-completion" \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "code": "function calculateSum(a, b) {", "language": "javascript", "context": "math-functions" }'
POST /v1/ai/code-review
Get AI-powered code review and suggestions
curl -X POST "https://api.azorahub.com/v1/ai/code-review" \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "code": "const result = calculateSum(1, 2);", "language": "javascript", "check_types": ["security", "performance", "style"] }'

Authentication

Personal Access Tokens

Use Personal Access Tokens (PATs) to authenticate your API requests. PATs are the recommended method for most use cases.

# Generate a new token curl -X POST "https://api.azorahub.com/v1/tokens" \ -H "Authorization: Bearer YOUR_MASTER_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "name": "My App Token", "scopes": ["repo", "read:org", "user"], "expires_at": "2024-12-31T23:59:59Z" }' # Use the token in API requests curl -X GET "https://api.azorahub.com/v1/user" \ -H "Authorization: Bearer YOUR_PAT"

OAuth 2.0

For web applications, use OAuth 2.0 to authenticate users on their behalf.

# Authorization URL https://api.azorahub.com/oauth/authorize? client_id=YOUR_CLIENT_ID& redirect_uri=YOUR_REDIRECT_URI& scope=repo%20read:org& response_type=code # Exchange code for token curl -X POST "https://api.azorahub.com/oauth/token" \ -H "Content-Type: application/json" \ -d '{ "client_id": "YOUR_CLIENT_ID", "client_secret": "YOUR_CLIENT_SECRET", "code": "AUTHORIZATION_CODE", "redirect_uri": "YOUR_REDIRECT_URI" }'

Code Examples

JavaScript/Node.js

const { AzorahubAPI } = require('@azorahub/api-client'); const client = new AzorahubAPI({ token: 'your-token-here' }); // List repositories const repos = await client.repositories.list(); console.log(repos); // Create an issue const issue = await client.issues.create('repo-id', { title: 'New feature request', body: 'Please add this amazing feature', labels: ['enhancement'] });

Python

import azorahub client = azorahub.AzorahubAPI(token='your-token-here') # List repositories repos = client.repositories.list() print(repos) # Create an issue issue = client.issues.create('repo-id', { 'title': 'New feature request', 'body': 'Please add this amazing feature', 'labels': ['enhancement'] }) print(issue)

Go

package main import ( "context" "fmt" "github.com/azorahub/go-azorahub" ) func main() { client := azorahub.NewClient("your-token-here") // List repositories repos, err := client.Repositories.List(context.Background()) if err != nil { panic(err) } fmt.Println(repos) // Create an issue issue, err := client.Issues.Create(context.Background(), "repo-id", &azorahub.IssueRequest{ Title: "New feature request", Body: "Please add this amazing feature", Labels: []string{"enhancement"}, }) if err != nil { panic(err) } fmt.Println(issue) }
